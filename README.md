# Introduction

This is a Chrome extension that hides the "Edit image" button shown when hovering over images on x.com (Twitter).

![screenshot](screenshot.jpg)

**Note:** This extension does not prevent others from editing or using your images. It only hides the "Edit image" button in your own browser.

# Installation

This extension works on Chromium-based browsers such as Chrome and Edge.

You can install this extension from the [Chrome Web Store](https://chromewebstore.google.com/detail/lpbakpipaokdchoikhaelkmkjpihnmcj)

# How to Use

1. Install the extension from Chrome Web Store.
2. Open or refresh any [x.com](https://x.com) page.
3. Hover over an image; the "Edit image" button will no longer appear.
4. Done!
